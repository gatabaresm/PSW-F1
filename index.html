Propósito del sitio


Planear propuestas de sitios web interactivos haciendo uso de los fundamentos teóricos de introducción y planeación de sitios web para la concepción de proyectos web.

La Actividad consiste en:


1. Realice una página web en formato HTML, en la cual usted debe colocar:

  • Título: Presaberes UNAD 2021. (Color azul, use etiqueta <h1>).
  • Encabezado: con sus datos básicos: Nombre, apellido, grupo, cédula, Skype, teléfono celular, correo personal. (Color negro, use etiqueta <h2>)
  • Diseñar una tabla: donde describa las principales etiquetas HTML (mínimo 10). (Debe aplicarle estilos CSS) (Use las etiquetas <table>, <tr>, <td>) (No debe ser copiada de internet).
  • Diseñar una segunda tabla: donde describa los diferentes estilos que pueden aplicarse en CSS (Colors, Backgrounds, borders, margins, text, fonts, Height, width, links, lists, tables, position, Overflow, float) indicando la sintaxis adecuada para aplicar los estilos. (Debe aplicar estilos) (Use las etiquetas <table>, <tr; <td>) (No debe ser copiada de internet).

2. Crear una carpeta comprimida en Zip que contenga el archivo HTML donde realizó su actividad.
