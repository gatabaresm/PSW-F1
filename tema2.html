position La propiedad "position" Especifica el tipo de método de posicionamiento usado para un elemento
"static"
Los elementos HTML son posicionados estaticamente por defecto
Un elemento con "position: static" se posiciona de acuerdo con el flujo normal de la página

Este elemento tiene "position: static";
<div class="static">
Este elemento tiene "position: static";
</div>

En la hoja de estilos: estilos.css colocamos las instrucciones:

div.static {
position: static;
border: 3px solid green;
}
position La propiedad "position" Especifica el tipo de método de posicionamiento usado para un elemento
"relative"
Un elemento con "position: relative" se posiciona relativamente con su posición normal

Este elemento tiene "position: relative";
<div class="relative">
Este elemento tiene "position: relative";
</div>

En la hoja de estilos: estilos.css colocamos las instrucciones:

div.relative {
position: relative;
left: 60px;
border: 3px solid green;
}
position La propiedad "position" Especifica el tipo de método de posicionamiento usado para un elemento
"fixed"
Un elemento con "position: fixed" se posiciona relativamente con el viewport, significa que siempre estará en la misma posición aún cuando se suba o baje la página

Este elemento tiene "position: fixed";
<div class="fixed">
Este elemento tiene "position: fixed";
</div>

En la hoja de estilos: estilos.css colocamos las instrucciones:

div.fixed {
position: fixed;
bottom: 0;
right: 0;
width: 100px;
border: 3px solid green;
}
